# PROJECT.md — Mini Library System

## Зорилго
Номын санг удирдах жижиг систем. Ном, гишүүн, зээлийг
бүртгэж, хянах боломжтой REST API.

## Scope (хамрах хүрээ)
### Багтах зүйлс 
- Номын бүртгэл (нэмэх, засах, устгах, хайх)
- Гишүүний бүртгэл (нэмэх, засах, устгах)
- Зээлийн бүртгэл (авах, буцаах, хугацаа хэтрэлт)

### Багтахгүй зүйлс 
- Frontend UI (API only)
- Төлбөр тооцоо
- Email notification

## Үндсэн feature-үүд
1. Book inventory — CRUD + хайлт
2. Member management — CRUD
3. Loan tracking — зээл авах/буцаах, хугацаа хянах
